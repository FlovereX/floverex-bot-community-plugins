## FlovereX Community Plugins

This is the official repository for community-created plugins for the FlovereX Bot.

## How to Contribute

1. Fork this repository.
2. Create your plugin in the plugins / folder.
3. Submit a Pull Request.

## Plugin Guidelines

- All plugins must use Discord.js v14.
- Do not include sensitive API keys or secrets.
- Keep code clean and documented.

Note: Submitting a plugin here does not guarantee it will be added to the main FlovereX Bot. All submissions are reviewed by the FlovereX team for security and quality.
